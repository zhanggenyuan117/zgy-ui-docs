## Attributes

| 参数| 说明 |可选值|类型|默认值| 
|-----| ----|-----|---|-------|
| type| 类型 | 'primary', 'success', 'info', 'warning', 'danger', 'text'|string|- |
| plain|朴素按钮| - |boolean| false |
|round|是否圆角| - |boolean| false |
|circle|是否圆形| - |boolean| false |
|disabled|是否禁用| - |boolean| false |
